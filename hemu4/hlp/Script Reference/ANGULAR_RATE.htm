ANGULAR_RATE

This command defines an angular rate for the specified entity or articulated part.

Syntax

ANGULAR_RATE    entity_id    part_id    yaw    pitch    roll

Parameters

entity_id

An integer that specifies the entity to which the rate is applied. If the rate is to be applied to an articulated part, then this argument specifies the entity to which the part is attached.

part_id

An integer that specifies the articulated part to which the rate is applied. If the rate is to be applied to an entity, then this parameter must be set to -1.

yaw

A floating-point number that specifies the yaw component of the rate.

pitch

A floating-point number that specifies the pitch component of the rate.

roll

A floating-point number that specifies the roll component of the rate.

Remarks

This command overrides any previously defined angular rate for the specified entity or articulated part. This command does not override previously defined linear rates.
