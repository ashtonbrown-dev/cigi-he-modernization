ENTITY_PITCH_OFFSET

Sets the pitch of the specified child entity with respect to the parent's body coordinate system.

Syntax

ENTITY_PITCH_OFFSET    id    pitch

Parameters

id

An integer specifying the entity.

pitch

The desired pitch of the entity in degrees.

Remarks

The specified pitch is with respect to the parent's body coordinate system. If the entity is a child, use ENTITY_PITCH to set the pitch relative to the world coordinate system.
