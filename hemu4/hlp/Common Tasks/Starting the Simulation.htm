CIGI Host Emulator 2.1

Starting the Simulation

To start or resume the simulation from its current state, click the Run Exercise button on the main toolbar. All entities, including special effects and weather, resume their previous operations. Any packets queued by the CIGI Host Emulator while the simulation was frozen are sent to the IG at this time.
