MISSILE_HIT_EFFECT

This command adds the specified animation entity to the missile's hit effect list. The animation will be played when the missile reaches its target.

Syntax

MISSILE_HIT_EFFECT    missile_id    animation_id

Parameters

missile_id

An integer that specifies the entity ID of the missile.

animation_id

An integer that specifies the entity ID of the animation.

Remarks

Each hit effect is allowed to continue its animation sequence until its duration expires. If the animation is set to loop, then the animation will continue playing indefinitely until the IG receives an appropriate ENTITY_ANIM_STATE command.

Any given missile may have more than one hit effect.
