Deleting an Entity

To delete an entity, first select the entity from the Entity Tree, then click the Remove Entity button on the toolbar. The entity will be removed from the tree and from the IG's scene graph.

When an entity is deleted, its ID is recycled. When a new entity is added, the CIGI Host Emulator will assign it a recycled ID if possible. Recycled IDs are used in the order in which the entities were deleted. Thus, if Entities 2 and then 5 are deleted, the next two entities created will be assigned 2 and 5 in that order. The next entity would then be assigned 6, or whatever is the next available unused ID.
