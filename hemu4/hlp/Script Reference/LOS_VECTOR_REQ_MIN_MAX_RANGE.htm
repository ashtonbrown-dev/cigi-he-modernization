LOS_VECTOR_REQ_MIN_MAX_RANGE

Defines the length of the line of sight test vector. The test vector is bound by a minimum and a maximum range from the source point.

Syntax

LOS_VECTOR_REQ_MIN_MAX_RANGE    min    max

Parameters

min

A floating-point number specifying the minimum range (in meters) at which intersections are registered.

max

A floating-point number specifying the maximum range (in meters) at which intersections are registered.

Remarks

This command must be preceded by a LOS_VECTOR_REQ_BEGIN command and followed by a LOS_VECTOR_REQ_END command.
