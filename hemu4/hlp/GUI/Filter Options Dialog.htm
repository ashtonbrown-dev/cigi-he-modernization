CIGI Host Emulator 2.1

Filter Options Dialog

The Filter Options dialog allows the user to specify which packets are displayed in the watch pane of the Test window. The dialog is shown below.

The IG-to-Host tab page lists the packets that may be sent from the IG and corresponds to the IG-to-Host column of the watch window. The Host-to-IG tab page lists those packets that the CIGI Host Emulator might send to the IG and corresponds to the Host-to-IG column of the watch window. Checking a packet type on either of these tab pages will cause packets of the appropriate type to be listed in the watch window.
