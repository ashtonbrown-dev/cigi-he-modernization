ENTITY_ADD_WAYPOINT

Adds a waypoint to the specified entity's waypoint list. The entity flies to waypoints in the order in which they are added to the list.

Syntax

ENTITY_ADD_WAYPOINT    entity_id    waypoint_id    airspeed    accel    turn_rate

Parameters

entity_id

An integer specifying the entity.

waypoint_id

An integer specifying the waypoint. This ID is assigned by the ID parameter of either the WAYPOINT_ADD_ABSOLUTE or the WAYPOINT_ADD_RELATIVE command.

airspeed

The desired airspeed (knots) of the entity after it passes through the waypoint.

accel

The acceleration (m/s2) of the entity as it reaches the airspeed indicated by the airspeed parameter.

turn_rate

The turn rate (degrees per second) of the entity as it adjusts its yaw to the next waypoint.

Remarks

Waypoints can be created with either the WAYPOINT_ADD_ABSOLUTE or the WAYPOINT_ADD_RELATIVE command.
