CIGI Host Emulator 3

ENTITY_FLY_WAYPOINTS

Causes the specified entity to fly the route defined by its waypoint list. The entity will fly this route until the script is stopped or the ENTITY_FLY command is encountered.

Syntax

ENTITY_FLY_WAYPOINTS    entity_id

Parameters

entity_id

An integer specifying the entity.

Remarks

Waypoints may be added to the entity's waypoint list with the ENTITY_ADD_WAYPOINT command.
