ENTITY_INHERIT_ALPHA

Specifies whether the specified child entity inherits its parent's alpha.

Syntax

ENTITY_INHERIT_ALPHA    id    inherit

Parameters

id

An integer specifying the entity.

inherit

An integer specifying whether the alpha is inherited. May be one of the following values:

    0 — Not inherited

    1 — Inherited
