LINEAR_RATE

This command defines a linear rate for the specified entity or articulated part.

Syntax

LINEAR_RATE    entity_id    part_id    x    y    z

Parameters

entity_id

An integer that specifies the entity to which the rate is applied. If the rate is to be applied to an articulated part, then this argument specifies the entity to which the part is attached.

part_id

An integer that specifies the articulated part to which the rate is applied. If the rate is to be applied to an entity, then this parameter must be set to -1.

x

A floating-point number that specifies the X component of the rate.

y

A floating-point number that specifies the Y component of the rate.

z

A floating-point number that specifies the Z component of the rate.

Remarks

This command overrides any previously defined linear rate for the specified entity or articulated part. This command does not override previously defined angular rates.
