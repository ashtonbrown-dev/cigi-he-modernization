VIEWGROUP_COMPONENT_STATE

Sets the discrete state of a view group component.

Syntax

VIEWGROUP_COMPONENT_STATE    group_id    component_id    state

Parameters

group_id

An integer that specifies the view group to which the component belongs.

component_id

An integer that specifies the component.

state

An integer specifying the discrete state of the component.

Remarks

This command must be followed by a VIEWGROUP_COMPONENT_END_UPDATE command.
