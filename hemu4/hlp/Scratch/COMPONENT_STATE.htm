CIGI Host Emulator 3

COMPONENT_STATE

Sets the discrete state of a component.

Syntax

COMPONENT_STATE    component_class    instance_id    component_id    state

Parameters

component_class

An integer that specifies class of the component. See Section 4.1.4 in the CIGI 3 Interface Control Document for class values.

instance_id

An integer that specifies the object to which the component belongs. The type of this object is determined by the component_class parameter.

component_id

An integer that specifies the component.

state

An integer specifying the discrete state of the component.
