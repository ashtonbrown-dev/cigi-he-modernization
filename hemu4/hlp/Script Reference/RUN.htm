CIGI Host Emulator 3

RUN

Runs the simulation for the specified amount of time. Execution of the script will be paused during this period.

Syntax

RUN    sec

Parameters

sec

The number of seconds for which to run. If this value is zero (0), the simulation will run indefinitely until the user freezes the CIGI Host Emulator via the main toolbar, Fly window, or the joystick.

Remarks

Entities whose fly modes are set to Fly will move according to their respective attitudes and airspeeds. To procedurally set an entity's fly mode to Fly,  use the ENTITY_FLY command before calling RUN.
