ENTITY_AIRSPEED

Sets the airspeed of the specified top-level (non-child) entity.

Syntax

ENTITY_AIRSPEED    id    airspeed

Parameters

id

An integer specifying the entity.

airspeed

The desired airspeed of the entity in knots.

Remarks

The ENTITY_SPEED keyword can be used to set the entity's speed in m/s.
