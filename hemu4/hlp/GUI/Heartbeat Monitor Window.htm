CIGI Host Emulator 2.1

Heartbeat Monitor Window

To show the Heartbeat Monitor window, shown below, click the Heartbeat Monitor button on the main toolbar.

This window displays a history of the IG-to-Host message timing. The green line represents the average interval between successive  start-of-frame messages within a given sample. The blue lines represent the minimum and maximum intervals encountered during the sample.

The scroll bar will increase or decrease the range displayed.
