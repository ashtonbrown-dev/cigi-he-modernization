CD_SEGMENT

Defines a collision detection segment for the specified entity.

Syntax

CD_SEGMENT    entity_id    segment_id    x1    y1    z1    x2    y2    z2    mask

Parameters

entity_id

An integer specifying the entity.

segment_id

An integer identifying the collision detection segment.

x1

The position (in meters) of the first endpoint along the entity's X axis.

y1

The position (in meters) of the first endpoint along the entity's Y axis.

z1

The position (in meters) of the first endpoint along the entity's Z axis.

x2

The position (in meters) of the second endpoint along the entity's X axis.

y2

The position (in meters) of the second endpoint along the entity's Y axis.

z2

The position (in meters) of the second endpoint along the entity's Z axis.

mask

The material mask specifying the surface types against which the segment will be tested.
