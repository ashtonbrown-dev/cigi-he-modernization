CIGI Host Emulator 3

CELSPHERE_COMPONENT_DATUM_DOUBLE

Sets a user-defined datum of a celestial sphere component as a double-precision (64-bit) floating-point real number.

Syntax

CELSPHERE_COMPONENT_DATUM_DOUBLE    component_id    instance_id    datum_id    value

Parameters

component_id

An integer that specifies the component.

instance_id

An integer that specifies the parent object of the component.

datum_id

An integer from 1 to 5 specifying the datum field that will receive the low-order 32-bits of value.  The high-order 32-bits of value will be placed in the next consecutive datum field (i.e., datum_id + 1).

value

A 64-bit floating-point number.

Remarks

The Host Emulator packages component data using Little-Endian byte order.

This command must be followed by a CELSPHERE_COMPONENT_END_UPDATE command.
