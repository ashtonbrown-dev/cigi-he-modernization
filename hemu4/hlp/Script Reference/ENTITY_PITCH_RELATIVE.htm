ENTITY_PITCH_RELATIVE

Sets the pitch of the specified top-level (non-child) entity with respect to the indicated reference entity.

Syntax

ENTITY_PITCH_RELATIVE    id    ref_entity    pitch

Parameters

id

An integer specifying the entity.

ref_entity

An integer identifying the reference entity. This may be the same entity as that specified by id.

pitch

The pitch (in degrees) of the specified entity relative to the reference entity.

Remarks

If id and ref_entity refer to the same entity, then that entity will be oriented relative to its current pitch.
