CIGI Host Emulator 3

COMPONENT_DATUM_INT64

Sets a user-defined datum of a component as a 64-bit signed integer.

Syntax

COMPONENT_DATUM_INT64    component_class    instance_id    component_id    datum_id    value

Parameters

component_class

An integer that specifies class of the component. See Section 4.1.4 in the CIGI 3 Interface Control Document for class values.

instance_id

An integer that specifies the object to which the component belongs. The type of this object is determined by the component_class parameter.

component_id

An integer that specifies the component.

datum_id

An integer from 1 to 5 specifying the datum field that will receive the low-order 32-bits of value.  The high-order 32-bits of value will be placed in the next consecutive datum field (i.e., datum_id + 1).

value

A 64-bit signed integer.
