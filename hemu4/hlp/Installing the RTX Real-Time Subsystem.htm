Installing the RTX Real-Time Subsystem

The Real-Time version of the CIGI Host Emulator consists of a Win32 component and a real-time component. The real-time component runs under a real-time extension to Windows called RTX. RTX is available from VenturCom, Inc at http://www.vci.com.

Note: The Windows version of the CIGI Host Emulator does not require RTX.

The installation of RTX 5.1 is a two-step process. First, the base RTX product must be installed. Then, the real-time RT-TCP/IP protocol stack (version 1.1 or newer) must be installed.

RTX 5.2 and later includes TCP/IP support. This component is automatically installed with RTX.

For more information, refer to the documentation included with the version of RTX you are using.
