ARTPART_Y

Specifies the Y offset of an articulated part.

Syntax

ARTPART_Y    entity_id    part_id    y

Parameters

entity_id

An integer specifying the entity to which the articulated part belongs.

part_id

An integer specifying the articulated part.

y

A floating-point value specifying the desired Y offset.

Remarks

The Y offset is relative to the entity's body coordinate system.
