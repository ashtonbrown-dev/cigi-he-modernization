CIGI Host Emulator 3

COMPONENT_DATUM_FLOAT

Sets a user-defined datum of a component as a single-precision (32-bit) floating-point real number.

Syntax

COMPONENT_DATUM_FLOAT    component_class    instance_id    component_id    datum_id    value

Parameters

component_class

An integer that specifies class of the component. See Section 4.1.4 in the CIGI 3 Interface Control Document for class values.

instance_id

An integer that specifies the object to which the component belongs. The type of this object is determined by the component_class parameter.

component_id

An integer that specifies the component.

datum_id

An integer from 1 to 6 specifying the datum field to be set.

value

A 32-bit floating-point number.
