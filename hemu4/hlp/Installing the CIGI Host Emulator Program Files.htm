Installing the CIGI Host Emulator Program Files

To install the CIGI Host Emulator, simply create a directory and copy the program files into it. The first time the program is run, the Host Emulator will display the Setup dialog so that the network properties may be configured.

The program folder should contain a subfolder named Default. Edit the .def files within this folder to specify the default entity, view, and environment properties for the Host Emulator.
