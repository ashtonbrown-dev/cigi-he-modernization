GLAYER_CLOUD_TYPE

Specifies the cloud type of a global weather layer.

Syntax

GLAYER_CLOUD_TYPE    id    type

Parameters

id

An integer specifying the global weather layer.

type

An integer specifying the cloud type. May be one of the following values:

    0 — None

    1 — Altocumulus

    2 — Altostratus

    3 — Cirrocumulus

    4 — Cirrostratus

    5 — Cirrus

    6 — Cumulonimbus

    7 — Cumulus

    8 — Nimbostratus

    9 — Stratocumulus

    10 — Stratus

    11 - 15 — Other
