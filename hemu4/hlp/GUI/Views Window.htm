CIGI Host Emulator 2.1

Views Window

The CIGI Host Emulator can be used to override the Image Generator's view configuration. A default view configuration may be defined in the VIEWS.DEF file. This configuration may be overridden by the user through the Views window, which is shown below:

The Views window contains the following items:

  • View ID
    • Selects the view whose properties are to be displayed by this window.
  • Group
    • Specifies a group to which to assign the current view. The following values are available:
    None Specifies that the view should not be assigned to a group.
    1-7 Specifies the group number to which the view should be assigned
  • Type
    • Specifies the view type for the current view. Refer to the appropriate Image Generator configuration documents for information on the available view types and the type number assigned to each.
  • Attach to Entity
    • Specifies the reference entity to which the view or group should be attached. 
  • View/Group
    • Specifies whether the Position and Orientation properties apply to the view or the view group.
  • X Offset
    • Specifies the position of the view eyepoint along the reference entity's X axis. The reference entity is specified by the Attach to Entity parameter.
  • Y Offset
    • Specifies the position of the view eyepoint along the reference entity's Y axis. The reference entity is specified by the Attach to Entity parameter.
  • Z Offset
    • Specifies the position of the view eyepoint along the reference entity's Z axis. The reference entity is specified by the Attach to Entity parameter.
  • Camera Heading
    • Specifies the rotation of the view eyepoint about an axis parallel to the reference entity's Z axis. The reference entity is specified by the Attach to Entity parameter.
  • Camera Pitch
    • Specifies the rotation of the view eyepoint about an axis parallel to the reference entity's Y axis. The reference entity is specified by the Attach to Entity parameter.
  • Camera Roll
    • Specifies the rotation of the view eyepoint about an axis parallel to the reference entity's X axis. The reference entity is specified by the Attach to Entity parameter.
  • Right FOV
    • Specifies the angle, in degrees, at which the right side of the view frustum extends from the eyepoint.
  • Top FOV
    • Specifies the angle, in degrees, at which the top of the view frustum extends from the eyepoint.
  • Left FOV
    • Specifies the angle, in degrees, at which the left side of the view frustum extends from the eyepoint.
  • Bottom FOV
    • Specifies the angle, in degrees, at which the bottom of the view frustum extends from the eyepoint.
  • Near FOV
    • Specifies in meters the distance from the eyepoint to the near clipping plane.
  • Far FOV
    • Specifies in meters the distance from the eyepoint to the near clipping plane.
  • Pixel Replication
    • Specifies the pixel replication mode of the view or view group.
  • Mirror View
    • Specifies the mirroring mode of the view or view group.
  • Enable Tracker
    • Specifies whether the tracking device should be enabled.
  • Assign to View
    • Specifies whether the tracking device, if enabled, should be assigned to the current view or view group.
  • Boresight
    • Boresights the tracking device. 
  • Slew and Zoom Control
    • Allows the user to control slewing and zooming of the current view or view group.
