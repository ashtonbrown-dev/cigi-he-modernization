LOS_SEGMENT_REQ_MATERIAL_MASK

Specifies a mask that determines which surface material types will generate a line of sight response for this request.

Syntax

LOS_SEGMENT_REQ_MATERIAL_MASK    mask

Parameters

type

A 32-bit hexadecimal value specifying the material mask for the line of sight test segment.

Remarks

This parameter specifies a mask against which an intersected polygon's material code is tested. If the binary product of the material code and mask is zero, then the intersection will not generate a line of sight response.

This command must be preceded by a LOS_SEGMENT_REQ_BEGIN command and followed by a LOS_SEGMENT_REQ_END command.
