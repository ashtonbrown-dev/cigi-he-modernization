CIGI Host Emulator 3

GTERRAIN_COMPONENT_DATUM_UINT32

Sets a user-defined datum of a global terrain component as a 32-bit unsigned integer.

Syntax

GTERRAIN_COMPONENT_DATUM_UINT32    component_id    instance_id    datum_id    value

Parameters

component_id

An integer that specifies the component.

instance_id

An integer that specifies the parent object of the component.

datum_id

An integer from 1 to 6 specifying the datum field that will receive value.

value

A 32-bit unsigned integer.

Remarks

The Host Emulator packages component data using Little-Endian byte order.

This command must be followed by a GTERRAIN_COMPONENT_END_UPDATE command.
