VIEW_PIXEL_REP_MODE

Sets the pixel replication mode of the specified view.

Syntax

VIEW_PIXEL_REP_MODE    id    mode

Parameters

id

An integer specifying the view.

mode

An integer specifying the pixel replication mode. May be one of the following values:

    0 — None

    1 — 1 x 2

    2 — 2 x 1

    3 — 2 x 2
