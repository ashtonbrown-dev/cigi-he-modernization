CIGI Host Emulator 3

Loading a New Configuration

When the CIGI Host Emulator starts, it looks in the "Default" subfolder of the Host Emulator program folder for definition (*.DEF) files containing the default configuration. This configuration is applied to all new scenarios until a scenario file (*.SF3) is loaded or another configuration is specified.

To load a new configuration, select File|Load Configuration... from the main menu.

From the Select Folder window, select the folder containing the desired definition (*.DEF) files. The new configuration will be applied to subsequently created entities. Existing entities will not be affected. Terrain, environment, and view configurations are also not updated at this time.

When a new configuration is loaded, it will be saved to any scenario files saved from that point on.
