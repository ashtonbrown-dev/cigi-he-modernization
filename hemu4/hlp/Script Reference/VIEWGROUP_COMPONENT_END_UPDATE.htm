CIGI Host Emulator 3

VIEWGROUP_COMPONENT_END_UPDATE

This command signifies that a view component has been updated. The command should immediately follow a VIEWGROUP_COMPONENT_STATE or VIEWGROUP_COMPONENT_DATUM[...] command or a group of these commands.

Syntax

VIEWGROUP_COMPONENT_END_UPDATE    group_id    component_id

Parameters

group_id

An integer that specifies the view group to which the component belongs.

component_id

An integer that specifies the component

Remarks

This command commits changes to a view group component and forces the Host Emulator to send a Component Control or Short Component Control packet to the IG. This allows a script to modify multiple component attributes in the same CIGI packet.
