CIGI Host Emulator 3

ENTITY_COMPONENT_DATUM_UINT64

Sets a user-defined datum of an entity component as a 64-bit unsigned integer.

Syntax

ENTITY_COMPONENT_DATUM_UINT64    entity_id    component_id    datum_id    value

Parameters

entity_id

An integer that specifies the entity to which the component belongs.

component_id

An integer that specifies the component.

datum_id

An integer from 1 to 5 specifying the datum field that will receive the low-order 32-bits of value.  The high-order 32-bits of value will be placed in the next consecutive datum field (i.e., datum_id + 1).

value

A 64-bit unsigned integer.

Remarks

The Host Emulator packages component data using Little-Endian byte order.

This command must be followed by an ENTITY_COMPONENT_END_UPDATE command.
