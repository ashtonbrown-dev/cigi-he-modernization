Using a Joystick Peripheral

The physical joystick controls the entity selected in the entity tree. Use the joystick-input toolbar button to enable or disable physical input. The button remains pressed while joystick input is enabled. This setting is loaded from the Joystick section of the application configuration.

Axes and Throttle

In Fly mode, the left/right axis controls roll, the forward/back axis controls pitch, and the rudder axis controls yaw. Flight mode is selected only in the Fly window; joystick buttons do not change the flight mode.

A physical throttle uses soft takeover. Move the throttle close to the selected entity's current airspeed or collective. The status bar reports when the values match, and the Fly window slider turns green while the physical throttle owns the value. The full throttle travel commands 0 to 1200 knots for other entities, or 0 to 100 percent collective for rotorcraft. Physical joystick and keyboard throttle input cannot command a negative speed.

The POV hat increases or decreases airspeed or collective. It also stops at zero and cannot command a negative speed.
