LOS_SEGMENT_REQ_END

Signals the end of the line of sight segment definition and causes a Line of Sight Segment Request packet to be sent to the IG.

Syntax

LOS_SEGMENT_REQ_END

Parameters

None

Remarks

This command must be preceded by a LOS_SEGMENT_REQ_BEGIN command and any other commands needed to define the line of sight segment request.

See Also
