PAUSE_MESSAGE

Pauses execution of the script and displays the specified text string in the message window. The user may resume the script by clicking the Play button on the Test toolbar.

Syntax

PAUSE_MESSAGE    string

Parameters

string

The text to be displayed. The string must be enclosed within quotation marks ("...").
