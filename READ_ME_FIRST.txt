Support for MFC and and the multibyte character set has been removed in Visual Studio 2013. Therefore 
this version of the Host Emulation requires the installation of the Multibyte MFC Library add-on for 
Visual Studio 2013, vc_mbcsmfc.exe. This can be obtained from:
	https://www.microsoft.com/en-gb/download/details.aspx?id=40770
