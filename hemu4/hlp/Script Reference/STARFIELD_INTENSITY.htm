STARFIELD_INTENSITY

Sets the brightness of the starfield.

Syntax

STARFIELD_INTENSITY    brightness

Parameters

brightness

The desired brightness of the starfield. This value is expressed as a percentage of the starfield's maximum brightness.

Remarks

The starfield can be enabled or disabled with the STARFIELD_ENABLE keyword.
