GLAYER_ENABLE

Enables or disables atmospheric effects within a global weather layer.

Syntax

GLAYER_ENABLE    id    enable

Parameters

id

An integer specifying the global weather layer.

enable

An integer specifying the desired state of the weather layer. May be one of the following values:

    0 — Disabled

    1 — Enabled
