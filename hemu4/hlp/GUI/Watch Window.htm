CIGI Host Emulator 2.1

Watch Window

The watch window pane of the Test window displays details of the CIGI message exchange between the IG and Host. This window is shown below:

The contents of each CIGI message selected on the Filter Options dialog is displayed as it is sent to or received from the IG. The packets are arranged according to frame so that all IG-to-Host packets appear next to the packets that appear in the corresponding Host-to-IG message. Frames are differentiated by alternating background color. Packets are listed in the order in which they appear in the message.

The window contains the following columns:

  • dT

This column displays the interval between consecutive IG Start of Frame messages. This value is the time in milliseconds from the previous message.

Note: For the real-time version of the CIGI Host Emulator, the accuracy of this value depends upon the value of the HAL Timer Period parameter set in the RTX Properties applet found in the Windows Control Panel. Refer to the RTX 5.1 documentation for information on configuring the RTX properties.

  • Host Resp Time

This column displays the number of milliseconds required by the CIGI Host Emulator to respond to the IG Start of Frame message.

  • IG-to-Host

This column lists the packets contained within the IG Start of Frame message for the given frame.

  • Host-to-IG

This column lists the packets contained within the host's response message for the given frame.

The columns may be resized to display information that may extend beyond the visible text.

To clear the contents of this window, click the Clear button on the Test toolbar.
