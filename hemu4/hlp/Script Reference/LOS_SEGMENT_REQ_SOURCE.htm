LOS_SEGMENT_REQ_SOURCE

Defines the source position for the line of sight test segment. This position can either be specified in Geodetic coordinates or relative to an entity.

Syntax

LOS_SEGMENT_REQ_SOURCE    type    x    y    z    entity_id

Parameters

type

Specifies how the source position is defined. May be one of the following values:

    GEODETIC

    ENTITY

x

A floating-point number specifying the X component of the source point's position. If type is set to GEODETIC, then this parameter specifies to the Geodetic latitude of the point. If type is set to ENTITY, then this parameter specifies the position relative to the entity's local X coordinate system.

y

A floating-point number specifying the Y component of the source point's position. If type is set to GEODETIC, then this parameter specifies to the Geodetic longitude of the point. If type is set to ENTITY, then this parameter specifies the position relative to the entity's local Y coordinate system.

z

A floating-point number specifying the Z component of the source point's position. If type is set to GEODETIC, then this parameter specifies to the altitude of the point. If type is set to ENTITY, then this parameter specifies the position relative to the entity's local Z coordinate system.

entity_id

An integer specifying the entity from which an entity-relative source point is defined. This parameter is ignored if type is set to GEODETIC.

Remarks

This command must be preceded by a LOS_SEGMENT_REQ_BEGIN command and followed by a LOS_SEGMENT_REQ_END command.
