ENTITY_YPR_OFFSET

Sets the attitude of the specified child entity with respect to the parent's body coordinate system.

Syntax

ENTITY_YPR_OFFSET    id    yaw    pitch    roll

Parameters

id

An integer specifying the entity.

yaw

A floating-point number representing the desired relative yaw of the entity in degrees.

pitch

A floating-point number representing the desired relative pitch of the entity in degrees.

roll

A floating-point number representing the desired relative roll of the entity in degrees.

Remarks

The specified attitude is relative to the parent's body coordinate system. If the entity is a child, use ENTITY_YPR to set the attitude relative to the world coordinate system.
