MISSILE_TARGET

This command specifies the target for a missile entity. The missile will track this target in response to a MISSILE_FIRE command.

Syntax

MISSILE_TARGET    missile_id    target_id

Parameters

missile_id

An integer that specifies the entity ID of the missile.

target_id

An integer that specifies the entity ID of the new target.

Remarks

This command overrides any previously specified target for the given missile.
