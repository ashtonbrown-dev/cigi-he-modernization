CIGI Host Emulator 2.1

Creating a New Entity

To create a new entity, click the Add Entity button on the Entity Tree toolbar. In the New Entity window, enter the entity ID and select an entity type.

New Entity

Click OK to add the selected entity and close the window. To add several entities without reopening the window, double-click an entity type. The entity is added immediately, the window remains open, and the ID box advances to the next available ID. Continue double-clicking the same or different entity types, then click Cancel or close the window when finished. Entities already added by double-clicking remain in the scenario.

The New Entity window is resizable. Drag an edge or corner to show more entity types; the type list expands with the window.

The ID box initially contains the next available entity ID. If one or more entities have been deleted, a recycled ID is offered in the order in which the entities were deleted. Otherwise, the next unused ID is offered. An ID that is already in use cannot be added again.

Select the intended entity type when creating the entity. Because some IGs might not allow an entity's type to change spontaneously, changing the type later may require deleting and recreating the entity.

Selecting an Entity Type

When the Ownship is created, it is placed at the default position specified for the database in the DATABASES.DEF file. If no default position is specified, the Ownship is placed at the database origin specified in the file. Successive entities are placed in a 60-meter-diameter circle around the Ownship at intervals of approximately 45 degrees.
