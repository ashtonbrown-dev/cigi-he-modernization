ENTITY_YAW

Sets the yaw angle of the specified entity.

Syntax

ENTITY_YAW    id    yaw

Parameters

id

An integer specifying the entity.

yaw

A floating-point number representing the desired yaw of the entity in degrees.

Remarks

The specified yaw is with respect to the world coordinate system. If the entity is a child, use ENTITY_YAW_OFFSET to set the yaw relative to the parent's body coordinate system.
