Defining Weather Entity Types

Weather entity types are defined in the ENTITIES.DEF file as shown below:

ENTITY
{
	NAME           = "Cloud";
	TYPE           = 1000;
	CLASS          = WEATHER;

	WEATHER_ATTRIBUTES
	{
		ENABLE			= TRUE;
		SCUD_ENABLE		= FALSE;
		RANDOM_WIND_ENABLE	= TRUE;
		RANDOM_LIGHTNING_ENABLE	= TRUE;
		CLOUD_TYPE		= 1;		// See ICD for assignments
		SEVERITY		= 3;		// 1 - 5
		SCUD_FREQUENCY		= 20;		// percent
		COVERAGE		= 65;		// percent
		TRANSITION_BAND		= 150;		// meters
		BAROMETRIC_PRESSURE	= 10;
		HUMIDITY		= 80;
		TEMPERATURE		= 32;
		VISIBILITY		= 50;
		HORIZONTAL_WINDSPEED	= 50;
		VERTICAL_WINDSPEED	= -20;
		WIND_DIRECTION		= 45;
		AEROSOL_CONCENTRATION	= 90;
	}
}

A weather entity type is defined as any other entity type (see Defining Entity Types) and will additionally include a WEATHER_ATTRIBUTES block to define default weather properties. Weather definitions specify the following parameters:

  • ENABLE
    A Boolean value specifying whether the weather phenomenon is enabled by default.  Acceptable values are TRUE or FALSE, YES or NO, and 1 or 0.
  • SCUD_ENABLE
    A Boolean value specifying whether scud effects are enabled by default.  Acceptable values are TRUE or FALSE, YES or NO, and 1 or 0.
  • RANDOM_WIND_ENABLE
    A Boolean value specifying whether random wind behavior is enabled by default.  Acceptable values are TRUE or FALSE, YES or NO, and 1 or 0.
  • RANDOM_LIGHTNING_ENABLE
    A Boolean value specifying whether random lightning is enabled by default.  Acceptable values are TRUE or FALSE, YES or NO, and 1 or 0.
  • CLOUD_TYPE
    An integer specifying the cloud type. Cloud types are enumerated in Section 4.1.12 in the CIGI 3 Interface Control Document.
  • SEVERITY
    An integer from 0 to 5 (least to most severe) specifying the intensity of the weather.
  • SCUD_FREQUENCY
    Specifies the frequency of scud as a percentage.
  • COVERAGE
    Specifies the horizontal coverage as a percentage.
  • TRANSITION_BAND
    Specifies the vertical thickness of the upper and lower transition bands. 
  • BAROMETRIC_PRESSURE
    Specifies the atmospheric pressure within the entity.
  • HUMIDITY
    Specifies the humidity as a percentage.
  • TEMPERATURE
    Specifies the air temperature within the entity.
  • VISIBILITY
    Specifies the visibility range through the entity in meters.
  • HORIZONTAL_WINDSPEED
    Specifies the horizontal windspeed in meters per second.
  • VERTICAL_WINDSPEED
    Specifies the vertical windspeed in meters per second.
  • WIND_DIRECTION
    Specifies the horizontal direction (in degrees) from which the wind blows.
  • AEROSOL_CONCENTRATION
    Specifies the concentration of water, dust, or other particles suspended in the air in grams per cubic meter.
