ENTITY_POS_RELATIVE

Sets the position of the specified top-level (non-child) entity with respect to the indicated reference entity.

Syntax

ENTITY_POS_RELATIVE    id    ref_entity    bearing    range

Parameters

id

An integer specifying the entity.

ref_entity

An integer identifying the reference entity. This may be the same entity as that specified by id.

bearing

The bearing (in degrees) of the specified entity relative to the reference entity's yaw.

range

The flat distance in meters between the specified entity and the reference entity.

Remarks

range is measured as if the displacement vector between the two entities were projected onto a plane tangent to the Earth's surface. In other words, the range does not account for a difference in altitude between the two entities. To specify a relative altitude, use the ENTITY_ALTITUDE_RELATIVE command.

If id and ref_entity refer to the same entity, then that entity will be moved relative to its current position.
