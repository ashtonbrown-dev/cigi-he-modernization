MISSILE_FIRE

This command launches a missile entity.

Syntax

MISSILE_FIRE    missile_id

Parameters

missile_id

An integer that specifies the missile entity to be launched.

Remarks

This command should be preceded by a MISSILE_TARGET command.

The missile must initially be attached to a parent entity.
