CIGI Host Emulator 3

Introducing the CIGI Host Emulator

The CIGI Host Emulator is a high-performance PC-based Image Generator (IG) test utility.  It conforms to the Common Image Generator Interface (CIGI) 3 protocol standard and is designed to be used for development and testing of all CIGI-compliant IGs.

The CIGI Host Emulator leverages off of the strengths of the Windows-Intel platform.  These include low system cost, a familiar, user-friendly graphical interface, an abundance of compatible systems, and extensive hardware support, among others.

The CIGI Host Emulator software is made up of the following components:

  • The Graphical User Interface (GUI) module is a standard Windows application.  It runs as a Win32 process and acts as the sole point of control for the user.
  • The driver is responsible for updating the position and orientation of each entity, communicating with the IG, and managing mission function data requests and responses.  The driver is invisible to the user and is loaded and unloaded automatically by the GUI process. The driver runs as a high-priority Windows process.
  • Definition Files are text files that are used to configure the CIGI Host Emulator and to define databases, views, entities, components, etc.
